# iiko-integration-module

Модуль для интеграции с сервисом iiko — набор PHP-утилит и клиента для упрощённого взаимодействия с iiko API

## Описание
Модуль предоставляет удобный PHP-клиент и вспомогательные функции для интеграции с iiko — российской системой автоматизации ресторанов. Цель модуля — скрыть детали HTTP-запросов и работы с токенами, чтобы вы могли быстро интегрировать заказ/меню/статусы и т.п. в ваше приложение.

## Возможности
- Авторизация и обновление access token
- Получение списка организаций и точек
- Работа с номенклатурой и категориями
- Универсальный HTTP-клиент с обработкой ошибок и логированием

### В модуле представлены классы processor и loader
- Processor принимает данные из API iiko-cloud и обрабатывает для дальнейшей отправки в loader
- Loader принимает данные из процессора и "загружает" в нужное место. Для каждого сервиса пишется отдельный загрузчик

Сейчас есть загрузчики для слудующих сервисов:
- opencart 3.X.X
- пустая БД

## Требования
- PHP 7.0+

## Установка
- Скопируйте исходники в папку вашего проекта, например src/Integration/Iiko
- Заполните config.example.php реальными данными
